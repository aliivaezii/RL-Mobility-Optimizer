Concept & Technical Core

💡 Core Idea

MoveWise is a MaaS super‑app that uses RL to transform private car users into sustainable commuters — not by telling them what to do, but by making the green choice the easy, fun, and financially smart choice.

🧠 RL Reward Function
rt = −[ w₁·GC + w₂·E + w₃·Ψbehav + w₄·Φconstr ] + w₅·Rrev
GC = Generalized Cost · E = Emissions · Ψ = Behavioral Model · Φ = Constraints · R = Platform Revenue
🎯 4‑Layer Behavior Change

Layer 1: Nudges

Default green, social proof, loss framing, commitment devices, anchoring (hidden car cost €510 vs PT €55)

Layer 2: Gamification

Green points, leaderboards, streaks, challenges, carbon budget, badges & milestones

Layer 3: Economic

Insurance discounts for PT users, employer packages, dynamic pricing, referral bonuses

Layer 4: Education

True Cost Calculator, "Did You Know?" cards, monthly impact reports, community stories

🔄 Nudge Selection RL
Nudge*(i,t) = argmaxn∈𝒩 Q̂(sit, n; θ)
A second RL agent learns which nudge works best for each user — personalized, not one‑size‑fits‑all.

📊 QR Data Collection
Inspired by Denmark's Rejsekort:
• Tap‑In: QR scan at journey start → timestamp, GPS, mode
• Tap‑Out: QR scan at destination → arrival time, location
• Pricing: Third‑party fares (GTT, Trenitalia, Voi) with transparent service margin
• Multi‑modal: Sequential scans reconstruct intermodal trips

Car users attracted via: Insurance, parking finder, fuel price comparison, carbon tracker, e‑commerce booking
